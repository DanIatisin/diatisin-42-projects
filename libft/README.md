# Libft

## Description

Libft is the first project of the 42 curriculum. The goal is to recreate a set of standard C library functions and build a personal static library that can be reused in future projects.

The library contains:
- Reimplemented libc functions
- Additional utility functions
- Linked list functions

## Instructions

Compile the library with:

```bash
make
```

Available rules:

```bash
make
make clean
make fclean
make re
```

The compilation generates:

```bash
libft.a
```

To use the library:

```c
#include "libft.h"
```

## Library Overview

### Libc Functions
Functions for:
- Character checks
- String manipulation
- Memory management
- Data conversion
